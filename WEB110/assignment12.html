JavaScript

While completing the JavaScript and HTML5 tutorials from Mozilla, W3Schools, and other sources, I learned how JavaScript can dynamically interact with HTML elements to make web pages more interactive and responsive. By following the CodeMaven “Hello World” example, I practiced using the p tag with JavaScript to display a personalized message like “Hello [my name]!” and also learned how to use the Date() function to show the current date and time dynamically. Additionally, experimenting with the HTML5 canvas helped me understand how JavaScript can be used to draw shapes, text, and graphics directly on a web page. These activities deepened my understanding of how JavaScript connects logic, structure, and visual design, and I can apply these skills in future web development projects to create more engaging and user-friendly websites or interactive interfaces.
