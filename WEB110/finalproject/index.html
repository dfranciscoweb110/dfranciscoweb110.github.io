What is CSS?

CSS, or Cascading Style Sheets, was created to separate the structure of a webpage from its presentation. Before CSS, HTML alone controlled both content and design, which made websites harder to maintain and limited visual styling options. CSS allows designers and developers to control colors, fonts, layouts, and spacing independently of HTML, making webpages more flexible, consistent, and easier to update. By using CSS, websites can be visually appealing, responsive, and accessible across different devices and screen sizes.

CSS Rule

A CSS rule is the basic building block of a stylesheet. It tells the browser which elements to style and what styles to apply.

Structure of a CSS Rule

A rule has two main parts:

  1. Selector — chooses the HTML element(s) to style
  2. Declaration block — contains one or more property: value; declarations
