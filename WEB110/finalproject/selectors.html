Understanding Selectors

Selectors target the HTML elements you want to style.

Common Selectors

  1. Element selector – targets tags like p or h1
    • Description: Targets all HTML elements of a specific type. Useful for applying styles to every instance of an element.
    • Example: p { color: blue; } → All paragraphs will have blue text.
  2. Class selector – targets elements with a class: .classname
    • Description:Targets elements that have a specific class attribute. Allows styling multiple elements that share the same class.
    • Example: .highlight { background-color: yellow; } → All elements with class="highlight" will have a yellow background.
  3. ID selector – targets one unique element: #idname
    • Description: Targets a single element with a unique ID. IDs should only appear once per page.
    • Example: #header { font-size: 24px; } → The element with id="header" will have larger text.
  4. Attribute selector – targets elements by attributes
    • Description: Targets elements based on the presence or value of an attribute. Very useful for form inputs or custom attributes.
    • Example: [type="text"] { border: 1px solid gray; } → All elements with type="text" will get a gray border.
