Understanding Selectors

Selectors target the HTML elements you want to style.

Common Selectors

  • Element selector – targets tags like p or h1
  • Class selector – targets elements with a class: .classname
  • ID selector – targets one unique element: #idname
  • Attribute selector – targets elements by attributes
