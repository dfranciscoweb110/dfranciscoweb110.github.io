SQL Statements Activity

The following are single-table SELECT statements used in this activity:

  • SELECT * FROM CUSTOMERS;
  • SELECT CustomerName, City FROM CUSTOMERS;
  • SELECT CustomerID, Country FROM CUSTOMERS WHERE Country = 'USA';
  • SELECT City FROM CUSTOMERS ORDER BY City;

The following are the INSERT, UPDATE, and DELETE statements used in this activity:

  • INSERT INTO CUSTOMERS (CustomerName, City, Country) VALUES ('John Doe', 'New York', 'USA');
  • UPDATE CUSTOMERS SET City = 'Los Angeles' WHERE CustomerName = 'John Doe';
  • DELETE FROM CUSTOMERS WHERE CustomerName = 'John Doe';

Databases

While completing these activities, I learned how to write and use SQL statements to manage and manipulate data within a database. I now understand how to retrieve specific information using SELECT statements and how to modify data with INSERT, UPDATE, and DELETE commands. This knowledge will help me in future work or projects by allowing me to organize, maintain, and analyze data effectively in any environment that relies on databases.
